# Fringe Drifters Accessories
Converting Fringe Drifters accessories into D&D-style weapon stats.

## Format
Each accessory is documented with:
- Damage: [dice] [type]
- Properties: [heavy/light/etc]
- Industrial Use: [primary non-combat function]
